GET /api/markets
Mirrors CoinGecko's markets endpoint. Returns cryptocurrency market data including prices, market caps, and volume. Supports query parameters for filtering and pagination.

Query Parameters

vs_currency (default: usd), order (default: market_cap_desc), per_page (default: 100), page (default: 1), sparkline (default: false), price_change_percentage, ids, category

Response

[{"id": "bitcoin", "symbol": "btc", "name": "Bitcoin", "current_price": 45000, ...}]

GET /api/volatility
Calculates volatility metrics (VWATR - Volume-Weighted Average True Range) for cryptocurrency bags based on historical OHLCV data stored in Vercel Blob. Returns VWATR and ATR% for multiple lookback periods.

Query Parameters

type (default: vwatr), bag (default: top20_bag, options: top20_bag, superstar_bag, all_coins), periods (default: 7,14,30, comma-separated list of days, max: 30 days)

Response

{"type": "vwatr", "bag": "top20_bag", "periods": [7,14,30], "maxPeriod": 30, "timestamp": 1234567890, "data": [{"symbol": "btc", "results": [{"period": 7, "vwatr": 0.0234, "atrp": 2.34}, ...]}, ...]}
